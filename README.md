# DAS Repository

This repository contains code for the DAS (Data Acquisition System).

## Overview

This tool is designed to support data acquisition from serial and Modbus communication interfaces. Includes GUI developed using Tkinter.

## Features

- Serial Port Communication
- Modbus Support
- Real-time Plotting
- Data Logging

## Requirements

- Python 3+
- Required Packages: pyserial, pymodbus, matplotlib
